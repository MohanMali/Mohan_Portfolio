What I Do

Responsibilities & Expertise

Production App Support (L2)

Troubleshooting and resolving enterprise application issues. Ensuring high availability, resolving tickets under strict SLAs, and maintaining system health.

Linux Server Administration

Administering Linux servers (Ubuntu/RHEL). Managing system resources, tracking CPU/memory load, auditing security logs, and managing users, permissions, and packages.

Log Analysis & Troubleshooting

Tracing application workflows, identifying errors using tools like tail, grep, awk, and examining application-level logs to perform thorough Root Cause Analysis (RCA).

Nginx & Web Servers

Configuring and hosting web servers with Nginx. Managing reverse proxy setups, virtual hosts, SSL/TLS certificate installation (Let's Encrypt), load balancing, and performance tuning.

Database Operations

Executing data fixes, optimizing basic queries, and debugging issues on MySQL, PostgreSQL, and MariaDB environments. Performing manual backups and data migrations.

Scripting & Automation

Automating redundant system administration workflows and repetitive tasks using custom Shell scripts and Python (crontabs, log rotation, database monitoring, alerts).
