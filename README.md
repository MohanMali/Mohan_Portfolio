## 💻 Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/portfolio.git
   cd portfolio
   ```
2. Open `index.html` in your web browser.

## 📦 Deploying to GitHub Pages
To host this portfolio for free using GitHub Pages:
1. Create a public repository named `portfolio` on your GitHub account.
2. Initialize and push your code (see the instructions provided by the Git setup helper).
3. On GitHub, go to your repository's **Settings** > **Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Set the branch to `main` (or `master`) and directory to `/ (root)`.
6. Click **Save**. Within a few minutes, your site will be live!
